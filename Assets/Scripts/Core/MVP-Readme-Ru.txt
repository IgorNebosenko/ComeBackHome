Версия 1.0.0

1. Для работы требуется ZenJect
2. Структуры данных определены через проверку наличия ELECTRUM_CORE. Если Core подключен - то структуры данных будут браться из него
3. Плагин пока что определён для работы с IEnumerable, вариации для кастомных тасок - нет

Использование:
- Инсталлер который создаёт экземпляры ViewManager и PopupManager - должен в методе AutoRegisterViewAttribute.GetViews() - брать сборку где лежат вьюхи и презентеры, иначе он ничего не найдёт
- Префабы с вьюхами - должны быть в Resources, по умолчанию - в папке Resorces/Views, если путь в ресурсах - инной, то в AutoRegisterView аттрибуте - указать другой путь
- Презентеры поддерживают инжект через конструктор

В примерах - описан MvpSceneInstaller, а также на примерах префаба вьюхи и попапа - можно глянуть как должна быть устроенна связь
